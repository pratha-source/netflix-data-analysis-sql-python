# Netflix Content Analysis (SQL & Python)

## 📌 Project Overview
This project explores the Netflix dataset to uncover trends in content production across different countries and years. It demonstrates a full data pipeline: from raw CSV to a SQL database, followed by advanced querying and visualization.

## 🛠️ Tech Stack
- **Language:** Python 3.13
- **Database:** SQLite
- **Libraries:** Pandas, Matplotlib, Seaborn

## 📊 Key Insights
- **Global Leader:** The US remains the top content producer, followed by India.
- **Format Shift:** Analysis shows a significant increase in TV Shows relative to Movies over the last 5 years.

## 🚀 How to Run
1. Clone this repository.
2. Install dependencies: `pip install pandas matplotlib seaborn`
3. Open `netflix_project.ipynb` in VS Code and run all cells.
